## Canonical artifacts

This protocol treats the working state as artifacts. The main artifact types are:

- Context packet: bounded brief for a single session.
- Selection log (selection trace): inclusion/exclusion receipt for the packet.
- Update proposals: suggested durable updates, non-binding until approved.
- Durable registries: authoritative long-term records for a scope (entities, rules, decisions, boundary policy, etc.).

The normative edition defines required sections, headers, and acceptance criteria.
