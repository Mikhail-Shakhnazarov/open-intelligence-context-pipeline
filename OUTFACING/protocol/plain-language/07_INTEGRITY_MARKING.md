# 07 COPY/PASTE DRIFT CHECK (INTEGRITY MARKING) (plain-language mirror)

## Purpose

Text artifacts are frequently transported by copy/paste across editors and tool surfaces. Accidental changes happen: formatting, missing lines, altered bullets, truncated sections.

Integrity marking (copy/paste drift check) is a visible marking pattern that helps detect accidental drift. It is not tamper-proofing and it does not provide adversarial security.

## Practical meaning

If an artifact is marked as durable and intended for transport, it should carry a drift check so that changes can be detected and investigated.

Exact marking formats and acceptance criteria are defined in the normative edition.
