## THE PROSE DEFAULT // 01

Prose is the system's first language. Information structure is expressed within sentences through subordination, coordination, reference, and punctuation rather than externalized into layout. Terse style is density: information per token relative to structural overhead.

These rules also define the dial inventory for intrinsic signage: observable variation points that preserve meaning while altering surface form.

## FORMAL RULES // 02

### Cohesion

Use controlled repetition for key terms. Use demonstratives for backward reference to maintain continuity. Avoid scaffolding phrases when structure already expresses the relationship.

### Structural encoding

Colons signal elaboration; semicolons coordinate alternatives; subordinate conjunctions embed scope without indentation. Prefer parallel syntax for enumerations when items share structure.

### Formatting constraints

Prefer in-language structure over heavy markup for relational content. Use lists and tables for comparison, procedure, and random-access lookup.

## DIAL FAMILIES (MINIMAL) // 03

Dial families are declared as a table in a signing profile. A minimal inventory may include:

- punctuation semantics: colon vs period where elaboration is preserved;
- coordination vs subordination where meaning is preserved;
- controlled connector alternation where meaning is invariant.

The inventory is extended only through explicit patches and conformance updates.
