# 01 CORPUS MANIFEST (plain-language mirror)

This is a plain-language mirror of: `../normative/01_CORPUS_MANIFEST.md`.
This mirror is not authoritative. If any discrepancy exists, the normative edition governs.

## What this is

This protocol is a text-defined workflow for producing bounded context for language-model runtimes. It defines artifact types, labeling, boundary rules, procedures, and conformance claims so the workflow can be reproduced across tools and runtimes.

This protocol is published in two editions:

- Normative (authoritative): `../normative/`
- Plain-language mirror (readability): `./`
