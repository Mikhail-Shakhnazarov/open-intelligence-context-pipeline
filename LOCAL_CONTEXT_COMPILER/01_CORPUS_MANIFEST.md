## SCOPE AND BOUNDARIES // 04

This corpus governs the Local Context Compiler as a text-defined system: artifact categories, procedures, and boundary rules for compiling context packs. It is designed to be portable across model runtimes and toolchains.

This corpus does not govern operator-produced content itself; it governs the procedures by which such content is selected, bounded, and proposed for promotion into durable state.
