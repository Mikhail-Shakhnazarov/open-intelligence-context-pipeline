# LOCAL_CONTEXT_COMPILER

This folder contains a corpus-level projection: a governed, system-description-style specification for a **Local Context Compiler** (LCC).

This corpus is explicitly a **derivative work** of the Earmark Open Intelligence Protocol (OpenIP) kernel in `the-corpus/` and is licensed **CC BY-SA 4.0** (ShareAlike applies). See `LOCAL_CONTEXT_COMPILER/01_CORPUS_MANIFEST.md`.

The aim is to specify the compiler *as a system* (its artifacts, boundaries, procedures, and conformance claims), not to compile any operator context yet, and not to define a model-vendor “memory” feature.
