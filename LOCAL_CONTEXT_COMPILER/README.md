# LOCAL_CONTEXT_COMPILER

This folder contains a standalone protocol corpus: a governed, system-description-style specification for a local context pipeline.

This corpus is licensed CC BY-SA 4.0 (ShareAlike applies). It is historically derived from OpenIP v1.0 (February 2026) and related work, but it is specified to be executable without any upstream protocol text in context. See `LOCAL_CONTEXT_COMPILER/kernel/01_CORPUS_MANIFEST.md`.

The aim is to specify the pipeline *as a system* (its artifacts, boundaries, procedures, and conformance claims), not to compile any operator context by default, and not to define a model-vendor "memory" feature.
